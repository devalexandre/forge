# mscli
Cli for create microservices with golang

## Install

```bash
go install github.com/mauricioklein/msgo
```

# Commands

## Create Architecture

```bash
msgo init <name>
```
## Create Service

```bash
msgo  create -n <name> -t service
```

## Create Repository

```bash
msgo  create -n <name> -t repository
```

## Create Contract

```bash
msgo  create -n <name> -t contract
```
